# Analizador_L-xico-
Analizador léxico, sintactico y semantico del lenguaje C con FLEX y Qt

 Cómo compilar
1. Tener instalado Qt 6 y WinFlexBison
2. Abrir CMakeLists.txt con Qt Creator
3. Configurar kit Desktop Qt 6.x MinGW 64-bit
4. Ctrl+B para compilar

## Ejecutable
La carpeta Ejecutable contiene el .exe listo para correr en Windows.
